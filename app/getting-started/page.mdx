# Elementor Template Installation Guide

> Quickly install and customize Elementor templates with this step-by-step guide.

---

## 📁 Folder Structure

After unzipping the downloaded file, you’ll see:

![Folder Structure](https://d2cassets.b-cdn.net/DOC-Image/folder-structure.png)

- `Fluent-Forms/` – Ready-to-import form files  
- `Custom.CSS` – Optional custom styles  
- `main-json.zip` – Includes header, footer, home  
- `require-plugins.txt` – Required plugins list  
- `site-setting.zip` – Global site settings

---

## ✅ Requirements

- WordPress admin access  
- Elementor (Free & Pro) installed  
- Recommended theme: **Astra** or **Hello Elementor**

---

## 1️⃣ Install Elementor & Pro

- Go to `Plugins > Add New` → Search for **Elementor** → Install & Activate  
- Upload Elementor Pro ZIP via `Plugins > Upload Plugin`

---

## 2️⃣ Install Required Plugins

- Open `require-plugins.txt`  
- Search & install each plugin listed  
- Theme: Go to `Appearance > Themes > Add New` → Install **Astra**

---

## 3️⃣ Import Site Settings

- Go to `Elementor > Tools > Import/Export`  
- Upload `site-setting.zip` under the **Import** tab

![Import Settings](https://d2cassets.b-cdn.net/DOC-Image/upload-site-setting.png)

---

## 4️⃣ Add Custom CSS

**Option 1: WPCode Plugin (Recommended)**  
- Install WPCode → Go to `WPCode > Code Snippets > Add Snippet`  
- Choose **CSS Snippet**, paste code, and **Activate**

![Custom Snippet](https://d2cassets.b-cdn.net/DOC-Image/create-custom-snippet.png)

**Option 2:**  
- Go to `Appearance > Customize > Additional CSS`

---

## 5️⃣ Enable SVG Support

- Go to `Elementor > Settings > Advanced`  
- Toggle **Enable SVG Uploads**

![Enable SVG](https://d2cassets.b-cdn.net/DOC-Image/svg-enable.png)

---

## 6️⃣ Import Fluent Forms

- Install & activate **Fluent Forms**  
- Go to `Fluent Forms > Tools > Import`  
- Upload forms from `Fluent-Forms/` folder

![Import Fluent Form](https://d2cassets.b-cdn.net/DOC-Image/fluent-form-upload.png)

---

## 7️⃣ Import Templates

**Option A – Import All at Once:**  
- Go to `Templates > Saved Templates > Import Templates`  
- Upload `main-json.zip`

![Import Template](https://d2cassets.b-cdn.net/DOC-Image/import-group-template.png)

**Option B – Import Individually:**  
1. Create & publish a new page  
2. Click **Edit with Elementor**  
3. Use **Folder Icon** → **My Templates**  
4. Import files from the extracted `main-json` folder (e.g., `header.json`, `home.json`)  
5. Click **Insert**

---

## 🔁 Multi-Page Template?

Repeat the import process for each `.json` (About, Contact, Services, etc.)
